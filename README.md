# 능주고등학교 출석 관리 시스템

능주고등학교를 위한 종합 출석 관리 시스템입니다. 기숙사형 학교의 특성을 고려하여 제작되었습니다.

## 주요 기능

### 1. 학년/반 관리
- 1학년부터 3학년까지, 각 학년당 2반 지원
- 반별 학생 관리 및 번호 관리

### 2. 학기/방학별 출석부
- 1학기, 2학기, 여름방학, 겨울방학별 출석부 관리
- 날짜별 교시 범위 설정 (1교시 ~ 12교시)
- 주말 출석부 지원 (기숙사형 학교 특성)

### 3. 교시별 시간 관리
- 각 교시별 시작/종료 시간 설정
- 시간 수정 가능
- 교시별 시간이 출석부에 표시됨

### 4. 출석 상태 관리
- **출석**: 정상 출석
- **결석**: 결석 (홈스쿨링 제외)
- **지각**: 지각
- **조퇴**: 조퇴
- **외출**: 외출
- **병결**: 병결
- **홈스쿨링**: 홈스쿨링 (결석률 미포함)
- **귀가**: 자율학습 안하고 집으로 가는 경우

### 5. 홈스쿨링 학생 관리
- 홈스쿨링 학생 선택 가능
- 홈스쿨링 기간 설정 (시작일 ~ 종료일)
- 해당 기간 동안 자동으로 홈스쿨링 상태로 처리
- **결석률 계산 시 홈스쿨링은 제외됨**

### 6. 귀가 학생 관리
- 자율학습을 하지 않고 집으로 가는 학생 표시
- 귀가 상태로 출석 체크 가능

### 7. 출석 현황 대시보드
- 한눈에 보기 쉬운 출석 현황 시각화
- 전체 통계 및 학생별 상세 통계
- 출석률 그래프 및 색상 코딩
- 날짜 범위 필터링

### 8. 데이터 저장
- 로컬 스토리지를 사용한 데이터 저장
- 브라우저를 닫아도 데이터 유지

## 설치 및 실행

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 빌드
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 사용 방법

### 1. 학생 등록
1. "학생 관리" 메뉴로 이동
2. "학생 추가" 버튼 클릭
3. 학생 정보 입력 (이름, 학년, 반, 번호)
4. 홈스쿨링 학생인 경우 체크박스 선택 및 기간 설정
5. 귀가 학생인 경우 체크박스 선택

### 2. 출석부 사용
1. "출석부" 메뉴로 이동
2. 학기/방학, 학년, 반 선택
3. 날짜 선택 (이전/다음 버튼으로 날짜 이동 가능)
4. 해당 날짜의 교시 범위 설정 (시작 교시 ~ 종료 교시)
5. "이 날짜 범위 저장" 버튼으로 저장
6. 교시별 시간 설정 (필요시 수정)
7. 각 학생의 교시별 출석 상태 선택
8. "설정 저장" 버튼으로 전체 설정 저장

### 3. 출석 통계 확인
1. "출석 통계" 메뉴로 이동
2. 학년/반 필터 선택
3. 날짜 범위 설정
4. 전체 통계 및 학생별 상세 통계 확인

## 주요 특징

- **직관적인 UI**: 누구나 쉽게 사용할 수 있는 깔끔한 인터페이스
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **실시간 저장**: 출석 상태 변경 시 자동 저장
- **색상 코딩**: 출석 상태별 색상으로 한눈에 파악 가능
- **유연한 설정**: 날짜별, 교시별 맞춤 설정 가능

## 기술 스택

- **React 18**: 사용자 인터페이스
- **TypeScript**: 타입 안정성
- **Vite**: 빠른 개발 환경
- **React Router**: 페이지 라우팅
- **date-fns**: 날짜 처리
- **Lucide React**: 아이콘
- **LocalStorage**: 데이터 저장
